DIFFTIME
========
Calcula la cantidad de tiempo tiempo trabajado entre dos fechas de acuerdo a un horario de trabajo. El horario de trabajo se ingresa dentro de la función.

Esta función se utiliza otras dos funciones:
F_TABLE_DATE: devuelve una tabla de fechas que contiene las fechas contenidas entre una fecha inicio y fin.
ES_FERIADO: devuelve si una fecha es considerada un feriado o no.
